# Srinidhi Portfolio

Personal portfolio website for Srinidhi Kummari, built to showcase projects, skills, and contact information in a clean, modern single-page layout.

## Overview

This portfolio highlights:

- Intro and profile section
- Featured projects
- About and skills section
- Contact and social links
- Responsive design for desktop and mobile

## Tech Stack

- HTML5
- Tailwind CSS
- JavaScript

## Featured Projects

- Customer Churn Prediction
- V2V Communication Demo

## Run Locally

1. Clone the repository:

```bash
git clone https://github.com/srinidhi1705/Srinidhi-portfolio.git
```

2. Open the project folder:

```bash
cd Srinidhi-portfolio
```

3. Install dependencies:

```bash
npm install
```

4. Start Tailwind in watch mode:

```bash
npm run dev
```

5. Open `index.html` in your browser.
